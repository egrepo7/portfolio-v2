Poke Your Friends!

Overview

This was my first full stack web app built using Python in the Pylot framework. The project demonstrates the "poke" functionality of Facebook. The goal was to be able track of all user information as well as user poke interaction using mySQL database. Twitter bootstrap was used to do front-end related styling.

Functionalities

  • Login/Registration of users
  • Ability to poke other users
  • Table of all users and their info
  • Display of total poke interactions

Technologies

Python, Pylot, mySQL, HTML5, JavaScript, Twitter Bootstrap

Top 5 Photo Viewer

Overview

Full Stack Web Application built using MEAN stack to demonstrate a voting system for multiple users and their photos. The back-end of this project was completed and deployed onto AWS all within 4.5 hours. Front-end was completed later using the Materialize framework.

Functionalities

  • Allows users to upload photos via URL
  • Upvote and downvote options
  • Table with uploaded photos and uploader info
  • Logged user able to view and delete photos
  • Filter function for photos

Technologies

JavaScript, MongoDB, Express, Angular, Node.JS, HTML5, MaterializeCSS
